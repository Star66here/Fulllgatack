1 ถ้าเข้ามาแล้ว เอาไฟลืเพื่อนมาแล้วต้องการทำงานสามารถทำได้เลย
2 ต้แงตรวจเช็คว่างานที่ตัวเองจะอัพอยู่ที่ไหนให้ถูกต้อง ให้ใช้คำสัั่ง cd ตามด้วยชื่อไฟล์เช่น 

C:\Users\Pipop\OneDrive\เอกสาร\GitHub\Work> cd Fullgatack 
PS C:\Users\Pipop\OneDrive\เอกสาร\GitHub\Work\Fulllgatack>  
ในกรณีที่เข้าผิดที่ให้ใช้ทำสั่ง cd .. 

3 ใช้คำสั่ง git add .  ในการตรวจสอบไฟล์ทั้งหมด ในโฟรเดอ ถ้าอยากได้แบบไฟล์เดียวให้ใช้ git add แล้วตามด้วยชื่อไฟล์

4 ใช้ git commit -m " สำคัญต้องให้คำอธิบายในนี้หรือไรก็ได้ห้ามปล่อยว่าง  " 
ใช้ในการว่าบอก git ว่ากูจะอัพงานนนะ

5 มี 2 กรณี

    5.1 ถ้าเข้ามาครั้งแรกควรใช้ git pull origin main เพื่อบอกว่าเราจะรีรีโมทมัน
แล้วตามด้วย git push origin main เพื่ออัพงาน

    5.2 git push origin main ได้เลยถ้าเคยอัพไปแล้วรอบนึง


6 ถ้าทำตามมาได้ขนาดนี้แล้วก็ทำตาม 1-5 ใหม่ทุกรอบที่จะอัพน่าจะให้ได้  สำคัญมากๆๆๆๆๆๆๆ มึงต็องเซฟ งานดด้วยเป็นไปได้ตั้ง
 เซฟ auto เลยจะดีมาก แล้วก็ใช้คำสั่ง git status เช็คได้ว่างานที่มึงจะอัพเนี่ยเป็นไง nothing to commit, working tree clean ถ้าขึ้นงี้คือมึงยังไม่ได้เซฟงานจ้าามันหาไม่เจอ
